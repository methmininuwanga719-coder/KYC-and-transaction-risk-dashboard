# 📊 KYC-and-Transaction-Risk-Dashboard

## 📌 Project Overview

This project presents an **Anti-Money Laundering (AML) Compliance Dashboard** developed using **Microsoft Power BI**. The dashboard provides interactive insights into customer risk, transaction activities, and compliance monitoring through clear visualizations and key performance indicators (KPIs).

---

## 🎯 Objectives

- Monitor compliance-related KPIs
- Identify high-risk and sanctioned customers
- Analyze transaction patterns
- Visualize customer distribution by country
- Support data-driven compliance decisions

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- CSV Datasets

---

## 📈 Dashboard Features

- 📌 KPI Cards
  - Total Customers
  - Total Transactions
  - High-Risk Customers
  - Sanctioned Customers

- 📊 Transaction Analysis
- 🌍 Country-wise Customer Distribution
- ⚠️ Risk Level Analysis
- 📉 Compliance Monitoring
- 🔍 Interactive Filters and Slicers
